About the Course

The R Programming course, offered by Johns Hopkins University and taught by Roger D. Peng, Brian Caffo and Jeff Leek, provides how to program in R and how to use R for effective data analysis

Covers practical issues in statistical computing which includes programming in R, reading data into R, accessing R packages, writing R functions, debugging, profiling R code, and organizing and commenting R code.

Program

Week 1

  • Background, Getting Started and Nuts & Bolts

Covers the history of R and S, go over the basic data types in R, and describe the functions for reading and writing data

Week 2

  • Programming with R

Covers key topics like control structures and functions.

Week 3

  • Loop Functions and Debugging

Covers loop functions and debugging tools in R.

Week 4

  • Simulation & Profiling

Covers how to simulate data in R. Also shows how to collect detailed information on how your R functions are running and to identify bottlenecks that can be addressed. Finally, covers the str function.
